# Simple Library App
A **Simple Library** application built with **plain JavaScript, HTML, and CSS** as part of  **The Odin Project – JavaScript course**.

The goal of this project is to practice **object-oriented JavaScript**, **DOM manipulation**, and other skills without using any frameworks.

### Features

- Add new books using a modal form
- Edit existing books
- Remove books from the library
- Mark books as *Read / Unread*
- Live search by **title** or **author**
- Responsive layout (desktop & mobile)
- Fully client-side (no backend)
